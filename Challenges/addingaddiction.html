Challenge 1 - Adding Addiction!

In this challenge, you will take the input of 5 different numbers, along with a word indicating the operation (add, subtract, multiply, divide), and output the result.




Input

You will need to set up a scanner to take the input of exactly five numbers. You must use an array with the length of 5.

Also, ask for a word which must be ADD, MULTIPLY, SUBTRACT, or DIVIDE and store that in a String.

Compute

Add, multiply, subtract, or divide all of the numbers in the array (you can hard code this or you can use a loop) and store it in a variable.

Output

Output the value into the console!
